Board Game Scanner - Hack-A-Thon Project

The board game scanner allows you to scan the bar code on the box of a game. After scanning, you will see the title, an image of the game along with the minimum and maximum price it sold for. You then can look for locations to purchase that specific game. Finally you can have instant access to YouTube videos of reviews and game play to learn how to play the game.

The front end frame work with HTML5, CSS3, JavaScript-OOP and Ajax for GET and POST requests. Used PHP to communicate with the UPCitemdb API. Used quaggaJS API for scanning the game board bar codes. YouTube API to pull videos. Google Maps API to find locations to purchase the game.
